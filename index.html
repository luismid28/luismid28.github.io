DIFERENCIA ENTRE DATO E INFORMACIÓN

Un dato es la unidad mínima de información que representa, un valor, hecho o característica, un dato por sí solo no tiene significado pero cuando se le agrega un contexto el dato se convierte en información, por ejemplo: el número 24 no significa nada, pero si le agregas Juan tiene 24 años, ahora es información.

CONCEPTO DE ESTRUCTURA DE DATOS

Las estructuras de datos son una forma de organizar y almacenar datos para poder usarlos, acceder a ellos y modificarlos de manera eficiente. Un ejemplo son los arreglos.

TIPOS DE DATOS Y ALGORITMOS

Muchas cosas pueden ser datos, un número, una letra, un símbolo, incluso una pequeña señal eléctrica, a esa señal se le llama bit y es la base de la informática. Los bits solo pueden representar los valores 0 y 1, un byte es un conjunto de 8 bits, con los bytes puedes representar caracteres. Las computadoras cuentan en base a 2.

Un algoritmo es un conjunto ordenado de pasos que describen cómo resolver un problema o realizar una tarea. Los algoritmos permiten convertir datos en información mediante distintos procesos por eso son tan importantes.

En la tercera clase vimos la introducción a la notación Big O.

NOTACIÓN BIG O

La notación Big O se usa para describir qué tan eficiente es un algoritmo, indicando cómo crece el tiempo o la memoria que necesita cuando aumenta la cantidad de datos de entrada.

Tiempo constante O(1):Es un tipo de comportamiento donde el algoritmo realiza una cantidad fija de pasos , sin importar cuántos datos existan.

Crecimiento lineal O(n):Es cuando el número de pasos del algoritmo crece al mismo ritmo que los datos.

Crecimiento cuadrático O(n²):Es cuando el algoritmo realiza operaciones por cada dato con todos los demás datos.

Crecimiento logarítmico O(log n):Es cuando el algoritmo reduce el problema en cada paso, normalmente dividiéndolo en partes más pequeñas.
